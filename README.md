# Educação Conectada

Educação Conectada é um projeto web desenvolvido com o objetivo de explorar soluções digitais voltadas ao **ambiente educacional**, utilizando tecnologias modernas do ecossistema front-end e integração com backend como serviço.

O projeto foi desenvolvido com foco acadêmico e prático, buscando aplicar conceitos de **desenvolvimento web moderno**, organização de código e boas práticas.

---

## Objetivo do Projeto

* Desenvolver uma aplicação web voltada ao contexto educacional
* Aplicar conceitos modernos de desenvolvimento front-end
* Integrar a aplicação com serviços de backend para persistência de dados
* Consolidar conhecimentos em TypeScript, Vite e Tailwind CSS
* Organizar um projeto escalável e bem estruturado

---

## Funcionalidades

* Interface web moderna e responsiva
* Estrutura preparada para autenticação e banco de dados
* Organização modular de componentes
* Integração com backend utilizando Supabase
* Base preparada para expansão de funcionalidades futuras

---

## Tecnologias Utilizadas

### Front-end

* Tailwind CSS
* HTML5

### Backend / Serviços

* GitHub

---

## Estrutura do Projeto

* `src/` contém os arquivos principais da aplicação
* Organização por componentes e páginas
* Configuração de estilos utilizando Tailwind CSS
* Pasta `supabase/` para controle de migrações e estrutura do banco de dados

---

## Como Executar o Projeto

* Clone o repositório
* Instale as dependências do projeto
* Configure as variáveis de ambiente do Supabase
* Execute o servidor de desenvolvimento
* Acesse a aplicação pelo navegador

> Observação: este projeto pode exigir configuração manual do Supabase para funcionamento completo.
